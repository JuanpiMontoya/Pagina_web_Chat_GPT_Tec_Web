Lenguajes de programación
Los lenguajes de programación son sistemas formales diseñados para expresar instrucciones que pueden ser ejecutadas por una computadora. Esto es fundamental ya que permiten a los programadores comunicarse con las máquinas para la creacion de software sea este en gran o en pequeña escala.
  • Python: Es un lenguaje de programación el cual presenta estructuras de datos de alto nivel eficientes y un sistema de programación orientado a objetos.Es ampliamente utilizado en las aplicaciones web, el desarrollo de software, la ciencia de datos y el machine learning
  • C++: Es un lenguaje de programación cuyo origen surgio como una extensión orientada a la manipulacion de objetos del lenguaje C y añadir mecanismos que permiten la manipulación de los mismos.
  • C#: Es un lenguaje de programación basado en objetos el cual permite a los desarrolladores crear muchos tipos de aplicaciones seguras y sólidas que se ejecutan en . NET
    • Bases de datos
    Una base de datos se encarga de la recopilación de datos sistemática y almacenada electrónicamente. Puede contener cualquier tipo de dato, incluidos palabras, números, imágenes, vídeos y archivos.
  • MySQL: Es un sistema de gestión de bases de datos relacional considerada como la base de datos de código abierto más popular del mundo utilizada sobre todo para entornos de desarrollo web.
  • MongoDB: es un sistema de base de datos NoSQL, orientado a documentos y de código abierto el cual guarda estructuras de datos BSON (similar a JSON) con un esquema dinámico.
  • Postgres: Es un sistema de gestión de bases de datos relacional orientado a objetos y de código abierto con una sólida reputación por su fiabilidad, flexibilidad y soporte de estándares técnicos abiertos.
    • Frontend
    El front-end se refiere a la práctica de construir y diseñar la interfaz de usuario de un sitio web utilizando las tecnologías adecuadas como HTML, CSS y JavaScript lo cual permite a los usuarios interactuar con la pagina web y que sea visualmente atractiva.
  • HTML: Es el lenguaje de marcado utilizado en la creación de páginas web donde se estructuran y desplegan sus contenidos.
  • CSS: Es un lenguaje informático que especifica cómo se presentan los documentos web a los usuarios: el como se diseñan, compaginan, etc.
  • JS: Es un lenguaje de programación orientado a objetos que se utiliza para hacer páginas web interactivas.
